## 这是啥

这是一个已经搭好的个人博客模板，包含首页、文章页、关于页、标签页、404 页面，以及自动部署流程。

## 站点特性

- 视觉风格偏安静、克制，但不是白底文档感。
- 首页强调作者气质和最新文章，不堆功能。
- GitHub Actions 自动部署到 GitHub Pages。
- 手机和桌面都有适配。

## 如果你是来看博客

请直接打开在线站点：

https://smithereensun.github.io/

## 本地开发

```bash
hugo server -D
```

## 部署方式

1. 推送到 `main` 分支。
2. GitHub Actions 自动构建。
3. GitHub Pages 自动发布。

## 可以继续改的地方

- `hugo.toml` 里的标题、介绍、联系方式。
- `content/posts/` 下的文章。
- `content/about.md` 里的个人介绍。
- `static/css/site.css` 里的视觉风格。
