实现步骤：先创建序列，后创建触发器

## 1、创建序列

```text
 create sequence 序列名
 increment by 1
 start with 1
 maxvalue 999999999;
```

## 2、创建触发器

```text
 create or replace trigger 触发器名
   before
      insert on 表名
   for each row
 declare
   -- local variables here
 begin
   SELECT 序列名.Nextval  INTO :NEW.自增列 FROM DUAL;
 end 触发器名;
```

##  3、创建键(UK)

```text
ALTER TABLE 表名
ADD CONSTRAINT 键名称
UNIQUE (列1, 列2, ... , 列n);
```

## 4、查询指定表触发器

```text
select * from all_triggers WHERE upper(table_name)=upper('表名')
```
