#  环境准备

## nodeJs

- [https://nodejs.org/](https://nodejs.org/)
- [https://nodejs.org/zh-cn/download](https://nodejs.org/zh-cn/download)

## git

- [https://git-scm.com/install/mac](https://git-scm.com/install/mac)

## 安装 Claude Code

windows

```text
打开命令行安装

npm install -g @anthropic-ai/claude-code
```

mac

```text
打开终端

npm install -g @anthropic-ai/claude-code
```
