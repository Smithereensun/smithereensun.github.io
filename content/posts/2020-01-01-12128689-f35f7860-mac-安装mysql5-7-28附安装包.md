## 配置环境变量 

### 在终端切换到根目录，编辑./.bash_profile文件

![](/imported/posts/2020-01-01-12128689-f35f7860-mac-安装mysql5-7-28附安装包/images/img_003_94f116234ba3.png)

```text
$ cd ~
$ vim ./.bash_profile
```

### 进入vim 编辑环境。 按下i 进入 insert 模式 ，输入

![](/imported/posts/2020-01-01-12128689-f35f7860-mac-安装mysql5-7-28附安装包/images/img_004_19c9fb7ee078.png)

```text
export PATH=$PATH:/usr/local/mysql/bin
export PATH=$PATH:/usr/local/mysql/support-files
```

### 按下esc 退出 insert 模式，输入:wq保存配置文件。

![](/imported/posts/2020-01-01-12128689-f35f7860-mac-安装mysql5-7-28附安装包/images/img_005_b7ef5967c569.png)

```text
:wq
```

### 在终端界面下输入以下命令，让配置文件的修改生效，并查看环境变量是否设置成功

![](/imported/posts/2020-01-01-12128689-f35f7860-mac-安装mysql5-7-28附安装包/images/img_006_cfca8ef9dd41.png)

```text
$ source ~/.bash_profile
$ echo $PATH
```

## MySQL服务的启停和状态的查看

![](/imported/posts/2020-01-01-12128689-f35f7860-mac-安装mysql5-7-28附安装包/images/img_007_403dec98fcee.png)

```text
停止MySQL服务
sudo mysql.server stop

重启MySQL服务
sudo mysql.server restart

查看MySQL服务状态
sudo mysql.server status
```

## 启动

### 终端界面下输入

```text
sudo mysql.server start
```

### 启动mysql服务,启动成功后继续输入

```text
mysql -u root -p
```

### 直接回车进入数据库，看到下列欢迎页面

![](/imported/posts/2020-01-01-12128689-f35f7860-mac-安装mysql5-7-28附安装包/images/img_008_78395bc03cd8.png)

### 若执行成功，初始化密码

　　设置初始化密码，进入数据库mysql数据库之后执行下面的语句，设置当前root用户的密码为root。

```text
set password = password('root');
```

![](/imported/posts/2020-01-01-12128689-f35f7860-mac-安装mysql5-7-28附安装包/images/img_009_41d506961ff5.png)

## mysql -u root -p登录失败用以下方法

### 首先第一步 停止Mysql服务 打开系统偏好设置中的mysql选择Stop Mysql Server

![](/imported/posts/2020-01-01-12128689-f35f7860-mac-安装mysql5-7-28附安装包/images/img_010_f72d10dc4da3.png)

### 进入终端输入：cd /usr/local/mysql/bin/ 回车后;

登录管理员权限 sudo su 回车后;输入管理员对应的密码

输入以下命令来禁止mysql验证功能 ./mysqld_safe --skip-grant-tables &  回车后mysql会自动重启（偏好设置中mysql的状态会变成running）

### 输入命令 ./mysql 回车后

![](/imported/posts/2020-01-01-12128689-f35f7860-mac-安装mysql5-7-28附安装包/images/img_011_db4881b14ad4.png)

### 设置密码 

输入命令 SET PASSWORD FOR 'root'@'localhost' = PASSWORD('你的新密码');

### 如果报错，提示信息如下

ERROR 1290 (HY000): The MySQL server is running with the --skip-grant-tables option so it cannot execute this statement

#### 先执行

```text
flush privileges;
```

#### 在修改密码

```text
alter user 'root'@'localhost' identified by '密码';
```
