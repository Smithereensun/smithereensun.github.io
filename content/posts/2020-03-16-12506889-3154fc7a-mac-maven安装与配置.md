# 配置环境变量 

## 总步骤

![](/imported/posts/2020-03-16-12506889-3154fc7a-mac-maven安装与配置/images/img_002_361252246a1a.png)

## 编辑.bash_profile文件

![](/imported/posts/2020-03-16-12506889-3154fc7a-mac-maven安装与配置/images/img_003_dbcac30d93e2.png)

```text
vim ~/.bash_profile
```

## 配置maven文件地址

![](/imported/posts/2020-03-16-12506889-3154fc7a-mac-maven安装与配置/images/img_004_cc6be64380e4.png)

```text
export M2_HOME=下载maven绝对路径
export PATH=$PATH:$M2_HOME/bin
```

## 保存文件，执行如下命令使配置生效

![](/imported/posts/2020-03-16-12506889-3154fc7a-mac-maven安装与配置/images/img_005_9eaac2362b49.png)

```text
source ~/.bash_profile
```

## 验证

![](/imported/posts/2020-03-16-12506889-3154fc7a-mac-maven安装与配置/images/img_006_2635e350995e.png)

```text
mvn -v
```
