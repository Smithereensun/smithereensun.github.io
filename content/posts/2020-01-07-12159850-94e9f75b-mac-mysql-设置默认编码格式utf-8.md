# 导读

　　博主百度一番，发现更改mysql默认编码格式，归结以下几个步骤。

# 详细步骤

## 切换当前目录

```text
cd /
cd private/etc
```

![](/imported/posts/2020-01-07-12159850-94e9f75b-mac-mysql-设置默认编码格式utf-8/images/img_001_ce4a4df1a8cf.png)

## 新建my.cnf文件

　　在当前目录下：private/etc

```text
sudo vim my.cnf

然后输入当前电脑的登录密码
```

![](/imported/posts/2020-01-07-12159850-94e9f75b-mac-mysql-设置默认编码格式utf-8/images/img_002_4217caba82fa.png)

## 输入以下内容

注：进入到编辑界面，先按“a”，将以下内容粘贴进去后，接着按“:”，然后输入“wq”

```text
[client]
default-character-set=utf8

[mysql]
default-character-set=utf8

[mysqld]
collation-server = utf8_unicode_ci
init-connect='SET NAMES utf8'
character-set-server = utf8
```

![](/imported/posts/2020-01-07-12159850-94e9f75b-mac-mysql-设置默认编码格式utf-8/images/img_003_091c21fb2239.png)

## 重启mysql服务

![](/imported/posts/2020-01-07-12159850-94e9f75b-mac-mysql-设置默认编码格式utf-8/images/img_004_c70bccec939f.png)

![](/imported/posts/2020-01-07-12159850-94e9f75b-mac-mysql-设置默认编码格式utf-8/images/img_005_9de5c16f10bb.png)

**搞定，以后默认编码格式为：utf-8**
