<h1 style="text-align: center">Vuex</h1>
<ul>
<li>
<p>定义</p>
<ul>
<li>Vuex 是集中式存储管理应用的所有组件的状态（数据）</li>
</ul>
</li>
<li>
<p>作用</p>
<ul>
<li>可实现任意组件之间的通讯</li>
</ul>
</li>
<li>
<p>特点</p>
<ul>
<li>当不同的组件需要对同一个状态进行读写时，或者复用的状态较多</li>
<li>能够保持数据和页面是响应式的</li>
<li>便于开发和后期数据维护</li>
</ul>
</li>
</ul>
<h2>安装</h2>
<div class="cnblogs_code">
<pre>cnpm install --save vuex</pre>
