# 本地生成ssh公私秘钥

```text
ssh-keygen
```

# 将本地公钥上传至服务器、

```text
本地公钥地址：C:\Users\xxxxx\.ssh\id_rsa.pub
先上传至服务器根目录：/home

创建文件夹：cd /home && mkdir .ssh

拷贝文件到.ssh：cp /home/id_rsa.pub /home/.ssh

修改文件名称：cd /home/.ssh && mv id_rsa.pub authorized_keys
```

最终是将本地公钥内容上传至linux文件中(**xxx可有可无**)：**/home/xxxx/.ssh/authorized_keys**

# 本地免密登录服务器

```text
ssh username@remote_host
```
