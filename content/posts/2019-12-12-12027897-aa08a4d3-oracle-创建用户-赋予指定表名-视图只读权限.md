# 步骤指南

## 创建用户

```text
格式：create user  TEST identified by 123456;

语法：create user  用户名 identified by 密码;

注：密码不行的话，前后加(单引号):'
create user  TEST identified by '123456';
```

## 授予connect权限

```text
格式：grant connect to TEST ;

语法：grant connect to 用户名 ;
```

## 授予只读权限

```text
格式：grant select on AAA.BBB to TEST;

语法：grant select on 表空间.表名(或视图名) to 用户名;
```
