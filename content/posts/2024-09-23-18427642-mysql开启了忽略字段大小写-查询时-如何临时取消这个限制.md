<p>解决办法：使用二进制比较函数，通过在查询中使用 <code>BINARY</code> 关键字进行二进制比较，这样 MySQL 会区分大小写</p>
<div class="cnblogs_code">
<pre><span style="color: rgba(0, 0, 255, 1)">SELECT</span> <span style="color: rgba(128, 128, 128, 1)">*</span> <span style="color: rgba(0, 0, 255, 1)">FROM</span> table_name <span style="color: rgba(0, 0, 255, 1)">WHERE</span> <span style="color: rgba(0, 0, 255, 1)">BINARY</span> column_name <span style="color: rgba(128, 128, 128, 1)">=</span> <span style="color: rgba(255, 0, 0, 1)">'</span><span style="color: rgba(255, 0, 0, 1)">Value</span><span style="color: rgba(255, 0, 0, 1)">'</span>;</pre>
