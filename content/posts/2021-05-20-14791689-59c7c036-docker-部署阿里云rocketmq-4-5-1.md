# 搜索镜像

```text
docker search rocketmq
```

# 查看镜像版本

如果要查看其它的镜像，只需要将其中的镜像名称foxiswho/rocketmq替换为其它镜像即可

```text
curl https://registry.hub.docker.com/v1/repositories/foxiswho/rocketmq/tags\
| tr -d '[\[\]" ]' | tr '}' '\n'\
| awk -F: -v image='foxiswho/rocketmq' '{if(NR!=NF && $3 != ""){printf("%s:%s\n",image,$3)}}'
```

# 启动nnameserver

```text
docker run -d -p 9876:9876 --name rmqserver  foxiswho/rocketmq:server-4.5.1
```

![](/imported/posts/2021-05-20-14791689-59c7c036-docker-部署阿里云rocketmq-4-5-1/images/img_002_3a3e9753f16f.gif)

# 启动broker

```text
docker run -d -p 10911:10911 -p 10909:10909\
 --name rmqbroker --link rmqserver:namesrv\
 -e "NAMESRV_ADDR=namesrv:9876" -e "JAVA_OPTS=-Duser.home=/opt"\
 -e "JAVA_OPT_EXT=-server -Xms128m -Xmx128m"\
 foxiswho/rocketmq:broker-4.5.1
```

# 进入broker容器，修改配置文件

　　注意修改自己云服务器公网ip

```text
docker exec -it 容器id /bin/bash
# 进入目录，找到配置文件broker.conf
cd /etc/rocketmq
# 修改broker.conf
vim broker.conf
# 在最后添加一行添加服务器公网IP
brokerIP1=47.116.143.16
```

# 重启broker

```text
docker restart xxx
```

# 启动ui控制台

注意修改自己云服务器公网ip

```text
docker run -d --name rmqconsole -e "JAVA_OPTS=-Drocketmq.namesrv.addr=47.116.143.16:9876 -Dcom.rocketmq.sendMessageWithVIPChannel=false" -p 8080:8080 -t styletang/rocketmq-console-ng
```
