<div class="cnblogs_code">
<pre>git log --format='%aN' | sort -u | while read name; do echo -en "作者： $name\t"; git log --author="$name" --pretty=tformat: --numstat | awk '{ add += $1; subs += $2; loc += $1 - $2 } END { printf "添加行数: %s, 删除的行数: %s, 代码总行数: %s\n", add, subs, loc }' -; done</pre>
