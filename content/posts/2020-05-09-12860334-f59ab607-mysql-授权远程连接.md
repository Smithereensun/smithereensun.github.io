# 解决方案

## 改表法

可能是你的帐号不允许从远程登陆，只能在localhost。这个时候只要在localhost的那台电脑，登入mysql后，更改 "mysql" 数据库里的 "user" 表里的 "host" 项，从"localhost"改称"%"
　　mysql -u root -pvmwaremysql>use mysql;
　　mysql>update user set host = '%' where user = 'root';
　　mysql>select host, user from user;

## 搜权命令

```text
grant 权限 on 数据库对象 to 用户
```

## 授权法

例如，你想myuser使用mypassword从任何主机连接到mysql服务器的话。
　　GRANT ALL PRIVILEGES ON *.* TO ['myuser'@'%'](mailto:'myuser'@'%') IDENTIFIED BY 'mypassword' WI
　　TH GRANT OPTION;

　　如果你想允许用户myuser从ip为192.168.1.6的主机连接到mysql服务器，并使用mypassword作为密码
　　GRANT ALL PRIVILEGES ON *.* TO ['myuser'@'192.168.1.3'](mailto:'myuser'@'192.168.1.3') IDENTIFIED BY;
　　'mypassword' WITH GRANT OPTION;
　　我用的第一个方法,刚开始发现不行,在网上查了一下,少执行一个语句 mysql>FLUSH RIVILEGES
　　使修改生效.就可以了
　　另外一种方法,不过我没有亲自试过的,在csdn.net上找的,可以看一下.
　　在安装mysql的机器上运行：
　　1、d:\mysql\bin\>mysql -h localhost -u root
　　//这样应该可以进入MySQL服务器
　　2、mysql>GRANT ALL PRIVILEGES ON *.* TO ['root'@'%'](mailto:'root'@'%') WITH GRANT OPTION;
　　//赋予任何主机访问数据的权限
　　3、mysql>FLUSH PRIVILEGES
　　//修改生效
　　4、mysql>EXIT
　　//退出MySQL服务器
　　这样就可以在其它任何的主机上以root身份登录啦
