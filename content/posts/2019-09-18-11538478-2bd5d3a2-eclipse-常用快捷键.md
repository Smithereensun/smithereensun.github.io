# 1、单行注释

```text
1 格式
2 注释：ctrl+/
3 取消：ctrl+/
```

# 2、多行注释

```text
1 格式
2 注释：ctrl+shift+/
3 取消：ctrl+shift+\
```

# 3、快速复制一行

```text
1 格式
2 向上复制一行：ctrl+alt+↑
3 向下复制一行：ctrl+alt+↓
```

# 4、向上移动代码

```text
1 格式
2 向上移动代码：alt+↑
3 向下移动代码：alt+↓
```

# 5、删除当前行

```text
1 格式
2 删除当前行：ctrl+d
```

# 6、格式化***
**

```text
1 格式
2 格式化：ctrl+shift+f
```

# 7、智能提示*****

```text
1 格式
2 智能提示：alt+/
```

# 8、选中一个变量全部修改

```text
1 双击变量名或者方法名
2 格式
3 alt+shift+r
```
