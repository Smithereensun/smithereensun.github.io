<p>第一步，依次点击，File - Settings - Bulid, Execution,Deployment - Compiler - Java Compiler，修改版本为13（你使用的java是哪个版本就选哪个喔）</p>
<p><img src="https://img2018.cnblogs.com/i-beta/1504448/202001/1504448-20200108200052839-18048134.png" alt="" /></p>
<p>&nbsp;</p>
<p>第二步，依次点击，File - Project Structure - Project，修改版本为13</p>
<p><img src="https://img2018.cnblogs.com/i-beta/1504448/202001/1504448-20200108200112093-1810729876.png" alt="" /></p>
<p>&nbsp;</p>
<p><img src="https://img2018.cnblogs.com/i-beta/1504448/202001/1504448-20200108200120851-1575891010.png" alt="" /></p>
<p>&nbsp;</p>
<p>第三步，还是在当前页，点击&nbsp;Modules，选择Sources选项，修改版本为13</p>
<p><img src="https://img2018.cnblogs.com/i-beta/1504448/202001/1504448-20200108200136294-131392183.png" alt="" /></p>
<p>&nbsp;</p>
<p>第四步，还是在当前页，选择Dependencies选项，修改版本为13</p>
<p><img src="https://img2018.cnblogs.com/i-beta/1504448/202001/1504448-20200108200150273-1150669143.png" alt="" /></p>
