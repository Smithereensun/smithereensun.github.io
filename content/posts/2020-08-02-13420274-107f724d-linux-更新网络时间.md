# 下载包

```text
yum install -y ntpdate
```

# 同步网络时间

```text
ntpdate 0.asia.pool.ntp.org
```

```text
若上面的时间服务器不可用，也可以改用如下服务器进行同步：
time.nist.gov
time.nuri.net
0.asia.pool.ntp.org
1.asia.pool.ntp.org
2.asia.pool.ntp.org
3.asia.pool.ntp.org
```

# 最后执行如下命令将系统时间同步到硬件，防止系统重启后时间被还原

```text
hwclock --systohc
```
