# 安装

## 下载

官网：[http://maven.apache.org/download.cgi](http://maven.apache.org/download.cgi)

![](/imported/posts/2020-09-13-13662849-7a6ff628-linux-maven安装/images/img_001_daeb43cb5c04.png)

## 解压

![](/imported/posts/2020-09-13-13662849-7a6ff628-linux-maven安装/images/img_002_58fb7015bf14.png)

```text
 tar -zxvf apache-maven-3.6.3-bin.tar.gz -C /usr/local/
```

## 修改环境变量

```text
vim /etc/profile
```

![](/imported/posts/2020-09-13-13662849-7a6ff628-linux-maven安装/images/img_003_13925aa04afd.png)

```text
配置文件生效：source /etc/profile
```
