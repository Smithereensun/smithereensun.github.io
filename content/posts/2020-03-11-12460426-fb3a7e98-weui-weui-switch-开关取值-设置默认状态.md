# html

```text
                <div class="weui-cell__ft">
                    <input class="weui-switch" type="checkbox" id="btn-switch">
                </div>
```

# js

```text
var str = $("#btn-switch").is(":checked");

注：选中返回值：true；未选中返回值：false
```

# 设置开关状态

## 默认开启

```text
                <div class="weui-cell__ft">
                    <input class="weui-switch" type="checkbox" id="btn-switch" checked="checked">
                </div>
```

## 默认关闭

```text
                <div class="weui-cell__ft">
                    <input class="weui-switch" type="checkbox" id="btn-switch"/>
                </div>
```
