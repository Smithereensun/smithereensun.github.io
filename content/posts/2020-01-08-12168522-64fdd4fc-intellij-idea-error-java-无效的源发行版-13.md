第一步，依次点击，File - Settings - Bulid, Execution,Deployment - Compiler - Java Compiler，修改版本为13（你使用的java是哪个版本就选哪个喔）

![](/imported/posts/2020-01-08-12168522-64fdd4fc-intellij-idea-error-java-无效的源发行版-13/images/img_001_7e16fe8acd58.png)

第二步，依次点击，File - Project Structure - Project，修改版本为13

![](/imported/posts/2020-01-08-12168522-64fdd4fc-intellij-idea-error-java-无效的源发行版-13/images/img_002_a02a88669ea6.png)

![](/imported/posts/2020-01-08-12168522-64fdd4fc-intellij-idea-error-java-无效的源发行版-13/images/img_003_bd572f8e682d.png)

第三步，还是在当前页，点击 Modules，选择Sources选项，修改版本为13

![](/imported/posts/2020-01-08-12168522-64fdd4fc-intellij-idea-error-java-无效的源发行版-13/images/img_004_9d1bdb3728b4.png)

第四步，还是在当前页，选择Dependencies选项，修改版本为13

![](/imported/posts/2020-01-08-12168522-64fdd4fc-intellij-idea-error-java-无效的源发行版-13/images/img_005_9df42f3ed46e.png)
