# 类注解

打开file->setting->Editor->File and Code Templates->Includes->File Header

![](/imported/posts/2020-07-29-13395026-6811f1ac-mac-idea-设置类注解说明/images/img_001_f3ad9d303188.png)

```text
#if (${PACKAGE_NAME} && ${PACKAGE_NAME} != "")package ${PACKAGE_NAME};#end
#parse("File Header.java")
/**
  * @ClassName：${NAME}
  * @Description：TODO
  * @Author：${USER}
  * @Date：${DATE} ${TIME}
  * @Versiion：1.0
*/
public class ${NAME} {
}
```
