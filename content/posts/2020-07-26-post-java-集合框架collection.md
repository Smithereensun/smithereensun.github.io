<p>集合容器主要用于保存对象，主要分类有三种List、Set、Map</p>
<ul>
<li>List有序、可重复的集合
<ul>
<li>常见的List有ArrayList、Vector、LinkedList等类</li>
</ul>
</li>
<li>Set无序、不可重复
<ul>
<li>常见Set接口的实现类有HashSet、LinedHashSet、TreeSet</li>
</ul>
</li>
<li>Map键值对存储
<ul>
<li>常见的Map接口实现类有HashMap和TreeMap</li>
</ul>
</li>
</ul>
<p>Collection接口有两个主要的子类List和Set，但Map不是Collection的子类，因为其本身就是一个顶层接口</p>
