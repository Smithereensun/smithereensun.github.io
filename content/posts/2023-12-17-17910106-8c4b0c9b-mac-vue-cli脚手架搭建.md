# 安装node环境

官网地址：[http://nodejs.cn/download/](http://nodejs.cn/download/)

我选择版本：v16.16.0

# 修改npm镜像地址

```text
# 查看镜像地址
npm config get registry

# 设置镜像地址
npm config set registry https://registry.npmmirror.com
```

# 安装vue

```text
cnpm install vue
```

# 安装vue-cli

```text
cnpm install -g @vue/cli
```

# 安装yarn

```text
# 安装
npm i -g yarn

# 查看版本
yarn -v
```
