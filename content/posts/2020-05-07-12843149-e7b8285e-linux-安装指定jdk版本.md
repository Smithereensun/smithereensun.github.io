# 操作步骤

## 卸载系统自带jdk版本

```text
1、查看安装的jdk
rpm -qa | grep java

2、卸载系统自带jdk
rpm -e --nodeps 包名
```

## 下载jdk

```text
　　当前最新版本下载地址：http://www.oracle.com/technetwork/java/javase/downloads/index.html

　　历史版本下载地址：　　http://www.oracle.com/technetwork/java/javase/archive-139210.html
```

这里我用的jdk1.8

```text
jdk1.8，百度云盘地址： https://pan.baidu.com/s/1DZGsJuLUrhpEQm7jaSKTwg  密码: baa5
```

## jdk国内镜像(下载速度快)

[点我直达](http://www.codebaoku.com/jdk/jdk-index.html)

## 解压

```text
1、cd xxxxxx要解压的目录

2、tar -zxvf 包名
tar -zxvf jdk-8u202-linux-x64.tar.gz
```

## 修改/etc/profile

最下面添加下面两行

```text
 export JAVA_HOME=/cyb/soft/jdk1.8.0_202
 export PATH=$JAVA_HOME/bin:$PATH
```

```text
export JAVA_HOME=/var/soft/jdk1.8.0_202
export PATH=$JAVA_HOME/bin:$PATH
export MAVEN_HOME=/usr/local/apache-maven-3.6.3
export PATH=$PATH:$MAVEN_HOME/bin
```

## 执行source操作

```text
source /etc/profile
```
