<h2>&nbsp;什么是HTTP协议？</h2>
<p>　　<span style="color: rgba(255, 0, 0, 1)"><strong>HTTP协议</strong></span>是建立在<span style="color: rgba(255, 0, 0, 1)"><strong>客户端和服务器</strong></span>之间的一个<span style="color: rgba(255, 0, 0, 1)"><strong>协议</strong></span>，在客户端和服务器之间需要数据的传输，而<span style="color: rgba(255, 0, 0, 1)"><strong>传输数据</strong></span>的时候，我们要<span style="color: rgba(255, 0, 0, 1)"><strong>按照指定的规则</strong></span>或者叫<span style="color: rgba(255, 0, 0, 1)"><strong>协议</strong></span>去<span style="color: rgba(255, 0, 0, 1)"><strong>传输数据</strong></span>。</p>
<p>　　HTTP是建立在<span style="color: rgba(255, 0, 0, 1)"><strong>TCP/IP</strong></span>协议基础之上的一个网络协议。</p>
<p>　　<span style="color: rgba(255, 0, 0, 1)"><strong>HTTP协议</strong></span>属于<span style="color: rgba(255, 0, 0, 1)"><strong>网络七层结构</strong></span>中最上层(<span style="color: rgba(255, 0, 0, 1)"><strong>应用层</strong></span>)的<span style="color: rgba(255, 0, 0, 1)"><strong>协议</strong></span>。</p>
<p>　　HTTP协议是一个无状态协议(不会记录每次访问时的信息)。</p>
<p>　　HTTP时一个<span style="color: rgba(255, 0, 0, 1)"><strong>客户端</strong></span>和<span style="color: rgba(255, 0, 0, 1)"><strong>服务器端</strong></span>请求和应该的标准(TCP)。客户端时终端用户，服务端时网站。</p>
<h2>什么是WEB资源？</h2>
<p>　　通过浏览器可以访问到的所有资源都是<span style="color: rgba(255, 0, 0, 1)"><strong>web资源</strong></span></p>
<p>　　WEB资源分为<span style="color: rgba(255, 0, 0, 1)"><strong>静态资源</strong></span>和<span style="color: rgba(255, 0, 0, 1)"><strong>动态资源</strong></span></p>
<p>　　　　• <span style="color: rgba(255, 0, 0, 1)"><strong>静态资源</strong></span>的数据是不变的，比如HTML、JPG、AVI。</p>
<p>　　　　• <span style="color: rgba(255, 0, 0, 1)"><strong>动态资源</strong></span>是通过后台程序展示页面数据的，比如Servlet请求。</p>
<h2>HTTP的作用是什么？</h2>
<p>　　为了<span style="color: rgba(255, 0, 0, 1)"><strong>约束客户端</strong></span>和<span style="color: rgba(255, 0, 0, 1)"><strong>服务器</strong></span>之间<span style="color: rgba(255, 0, 0, 1)"><strong>传输web资源时</strong></span>的<span style="color: rgba(255, 0, 0, 1)"><strong>格式</strong></span>。</p>

<p>　　HTTP请求协议信息由三部分组成：<span style="color: rgba(255, 0, 0, 1)"><strong>请求行、请求头、请求体</strong></span></p>
