## 编译安装

```text
解压:　　
    tar zxvf redis-4.0.14.tar.gz

移动到:　　
    mv redis-4.0.14.tar.gz /Users/chenyanbin/plus/

切换到:　　
    cd /Users/chenyanbin/plus/redis-6.0.9　　

编译测试:
    sudo make test

编译安装:　　
    sudo make install

启动
    redis-server&

测试
    redis-cli

停止
    redis-cli shutdown

查看是否启动
     ps axu|grep redis
登录指定redis
    redis-cli -h 127.0.0.1 -p 6379
```
