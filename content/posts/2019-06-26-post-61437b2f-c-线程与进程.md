<h1>一、前台线程与后台线程对象</h1>
<h2><span style="color: rgba(255, 0, 255, 1)"><strong>为什么要用多线程?</strong></span></h2>
<p><span style="color: rgba(255, 102, 0, 1); font-size: 18px"><strong>　　</strong><span style="color: rgba(0, 0, 0, 1)">1.让计算机“同时”做多件事情，节约时间。</span></span></p>
<p><span style="color: rgba(255, 102, 0, 1); font-size: 18px"><span style="color: rgba(0, 0, 0, 1)">　　2.多线程可以让一个程序“同时”处理多个事情。</span></span></p>
<p><span style="color: rgba(255, 102, 0, 1); font-size: 18px"><span style="color: rgba(0, 0, 0, 1)">　　3.后台运行程序，提高程序的运行效率，也不会导致主界面出现无响应的情况。</span></span></p>
<p><span style="color: rgba(255, 102, 0, 1); font-size: 18px"><strong>线程</strong></span></p>
<p>- 前台线程:只有所有的前台线程都关闭才能完成程序关闭。</p>
<p>- 后台线程：只有所有的前台线程结束，后台线程自动结束。(设置方法:线程实例名.IsBackground = true)</p>
