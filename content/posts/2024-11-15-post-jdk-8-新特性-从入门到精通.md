<h1 style="text-align: center">default关键字</h1>
<p>　　在<strong><span style="color: rgba(255, 0, 0, 1)">jdk1.8以前</span></strong>接口里面是<span style="color: rgba(255, 0, 0, 1)"><strong>只能有抽象方法</strong></span>，<strong><span style="color: rgba(255, 0, 0, 1)">不能有任何方法的实现的</span></strong>。</p>
<p>　　在<span style="color: rgba(255, 0, 0, 1)"><strong>jdk1.8里面打破</strong></span>了这个<span style="color: rgba(255, 0, 0, 1)"><strong>规定</strong></span>，<span style="color: rgba(255, 0, 0, 1)"><strong>引入</strong></span>了<span style="color: rgba(255, 0, 0, 1)"><strong>新</strong></span>的<span style="color: rgba(255, 0, 0, 1)"><strong>关键字</strong></span>：<span style="color: rgba(255, 0, 0, 1)"><strong>default</strong></span>，使用<span style="color: rgba(255, 0, 0, 1)"><strong>default修饰方法</strong></span>，<span style="color: rgba(255, 0, 0, 1)"><strong>可以在接口里定义具体的方法</strong></span></p>
