<ul>
<li>IoC（<span style="color: rgba(255, 0, 0, 1)"><strong>核心中的核心</strong></span>）：Inverse of Control，控制反转。<strong><span style="color: rgba(255, 0, 0, 1)">对象创建的权利由程序员交给Spring框架</span></strong></li>
<li>AOP：Aspect Oriented Programming，面向切面编程。在不修改目标对象的源代码情况下，增强IoC容器中Bean的功能</li>
<li>DI：Dependency Injection，依赖注入。在Spring框架负责创建Bean对象时，动态的将依赖对象注入到Bean组件中！！</li>
<li>Spring容器：就是IoC容器</li>
</ul>
