# 思路

　　单表导入的比较容易，但是有的时候，可能会出现，一对多数据导入的，这个情况怎么办呢？**先理解上面的图**，后台获取数据的时候，**除**了“**风险防控措施**”**外**，**其他字段先分组**，“**黄色背景**”、“**蓝色背景**”、“**绿色背景**”，此时，**一条“黄色背景”对应3条防控措施，一条“蓝色背景”对应1条防控措施，一条“绿色背景”对应1条防控措施**。
