## [题⽬描述](#题目描述)

给定⼀个句⼦，将句⼦中的单词进⾏翻转，注意，单词内部的字符顺序不改变，改变的是单词与单词之间的顺序，⽐如“ I am a coder. ”,翻转之后变成“ coder. a am I ”。

示例
 注意：引号内部才是输⼊的内容
 输⼊："You are a cool boy."
 输出："boy. cool a are You"

## [思路及解答](#思路及解答)

### [Java内置函数分割反转](#java内置函数分割反转)

Java ⾥⾯有切割字符串的⽅法，直接⽤空格进⾏分隔，分隔成为多个单词，然后从字符串数组的后⾯开始，每⼀个字符拼接上⼀个空格，但是，值得注意的⼀点是，当只剩下⼀个字符串的时候，也就是索引为 0 的时候，不需要再拼接空格。代码实现如下：

- **时间复杂度**：O(n)，分割和拼接各需线性时间

- **空间复杂度**：O(n)，存储单词数组和结果字符串

### [借助栈](#借助栈)
![](./images/images/2369cddfa9b7-202605202239304.png)
先使用空格将句子切分成单词数组，然后从左到右遍历单词。为了在最终拼接时保留单词之间的空格，前面的每个单词入栈后，都额外压入一个空格。最后一个单词后面不再压入空格，避免翻转后的结果开头出现多余空格。

当所有单词和空格都压入栈后，再不断 pop 出栈并追加到 StringBuilder 中。由于最后入栈的单词会最先弹出，所以单词顺序被反转，而每个单词内部的字符顺序保持不变。
![](./images/images/1319689475cf-202605202238119.png)![](./images/images/b68dc0ea5e6a-202605202239985.png)
### [双指针法(推荐)](#双指针法-推荐)

⾸先判断字符串 str 是否为空或者为空字符，如果 str 不为空，则初始化 start 和 end 指针指向字符串的尾部， start 从尾部向头部遍历

针对每⼀个字符，如果字符为空字符：

- 如果 start 和 end 不是处于同⼀个位置，说明已经遍历完⼀个单词，那么将 start+1 ~ end 之间的字符，拼接到结果后，并且添加上⼀个空格。将 start-1，并且 end 指向 start 的位置

- 否则，说明是多余的空格，直接跳过， start-1， end=start

如果字符不是空字符：可以把 start 索引前移，也就是 start--

最后，剩下最后⼀个单词的时候，需要特殊处理最后⼀个单词，拼接到字符串后⾯就可以了。
![](./images/images/2bd6e2a69466-202605231145477.png)

- **时间复杂度**：O(n)，每个字符被访问常数次

- **空间复杂度**：O(n)，StringBuilder存储结果

### [三次反转法（空间最优）](#三次反转法-空间最优)

通过字符级操作实现O(1)空间复杂度（假设字符数组可修改）。

先反转整个字符串，再逐个反转每个单词

**三次反转的数学原理：**

对于句子 "A B C"（A、B、C为单词）：

1. **整体反转**： "C' B' A'"

1. **单词反转**： "C'"反转→ "C", "B'"反转→ "B", "A'"反转→ "A"

1. 结果： "C B A" → 正是期望的顺序

![](./images/images/483a6192d68e-202605202242195.png)![](./images/images/2fff2f248d21-202605202242136.png)

- **时间复杂度**：O(n)，三次线性遍历

- **空间复杂度**：O(1)（如可修改原数组）或 O(n)（需创建新数组）
