# 验证环境

```text
# 1. 验证Java环境（Spring Boot 3.x需JDK 17+）
java -version
# 期望输出：包含 “java version “17.x.x”

# 2. 验证Maven（项目管理）
mvn -v
# 期望输出：包含 “Apache Maven 3.6+” 版本信息

# 3. 验证Node.js（Vue开发）
node --version
# 期望输出：包含 “v18.x.x” 或更高

# 4. 验证npm包管理器
npm -v
```
