# 导读

　　工欲善其事必先利其器，日常工作中，知道这些Idea技巧，可以极大提高日常开发效率。

## 搜索

1. 搜索方法，按两下shift
2. 文字搜索，control+shift+f；可以搜指定项目、目录下的文字、文件后缀等

## 代码格式化

　　快捷键：option+command+L

# 插件篇

## 打印SQL语句

　　如果使用的Mybatis框架，控制台打印SQL语句的时候，使用最多的是log4j，需要手动配置，这里我们可以使用：**Mybatis log plugin**

　　这个插件是要收费的，但是你懂得，嘿嘿(*^▽^*)，这里我有一个是idea2020.1的，安装步骤：[点我直达](https://www.cnblogs.com/chenyanbin/p/13782894.html)

![](images/img_024_8b4c9f808c10.gif)

## 翻译插件

　　对于一些英语不好的朋友，这个插件简直是太棒啦👍：**Translation**

![](images/img_025_bf82c7c982d6.gif)

## 代码质量检查

　　这个工具可以提高代码质量，避免不必要的错误发生，当然啦感兴趣的可以去研究下Sonarqube，这里我提供一个地址：[点我直达](https://www.cnblogs.com/chenyanbin/p/qq543210188.html)，我们可以使用这个插件：**Alibaba Java Coding Guidelines**

![](images/img_026_c5cbe1003201.gif)

## 生成所有的set属性

　　插件名字：**GenerateAllSetter**，当我们要为整个类生成setXXX的时候，可以利用该插件快速生成set方法。快捷键：option+回车

![](images/img_027_ac37d38fa23a.gif)

## 快捷键提示插件

　　插件名：**key promoter**，该插件，可以帮助我们记住快捷键

![](images/img_028_f74f68ae38b6.gif)

## lombok插件

　　插件名：**lombok**，这里就不过解释他的用法了，不会的自行百度

## GitToolBox

　　在Idea的状态栏显示git状态，还提供了定时fecth等功能。可以在每一行后面显示提交人和时间等信息，有了它可以避免吐槽到自己 
![](images/img_029_87c408e41433.gif)
